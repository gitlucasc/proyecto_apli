Bienvenidos a la Guía del Curso de 6°5 - Modalidad Informática

Este sitio está diseñado como una guía para los futuros alumnos que están por comenzar el curso de 6°5 en la modalidad informática. Aquí encontrarás toda la información que necesitas para prepararte para las clases teóricas y para el taller práctico. ¡Explora conocimientos en desarrollo web, programación y redes para aprovechar al máximo esta etapa!
