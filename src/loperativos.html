Laboratorio de Sistemas Operativos

En esta materia, los estudiantes estudian e implementan los conceptos de los sistemas operativos, así como los problemas que pueden presentar en entornos de red y servidor.

Contenidos:

Primer Cuatrimestre:

  • Diferencias entre arquitecturas de red igualitaria y jerárquica.
  • Planificación y configuración de redes: infraestructura, recursos, equipos y usuarios.
  • Selección de servidores adecuados a las demandas de hardware y software.
  • Comparación entre Windows Server y Linux Server en redes.
  • Instalación y configuración de Windows Server (2003-2008).
  • Configuración de conectividad entre el servidor y las estaciones de trabajo.

Segundo Cuatrimestre:

  • Instalación y configuración de funciones en el servidor.
  • Análisis y configuración del Servicio de Directorio Activo.
  • Gestión de dominio y subdominio, y agregar equipos al dominio.
  • Creación y administración de usuarios, grupos y unidades organizativas.
  • Uso de PHP para la conexión de aplicaciones web al servidor.
  • Configuración de políticas de seguridad en la red para gestión de permisos.

Trabajo Práctico Integrador Anual

El trabajo integrador constará de:

  • Una parte teórica donde el alumno deberá planificar la instalación de una red según un caso planteado.
  • Una parte práctica en la que pondrá en marcha un servidor y configurará los componentes necesarios.
